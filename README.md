# Lead2Lease

## Development Scripts

- `npm run dev`: Starts the full stack when working from Replit.
- `npm run dev:cursor`: Starts the same stack using `dotenv-cli` for Cursor workspaces.

Choose the script that matches the environment you’re developing in so environment variables load correctly.




How to Setup Lead2Lease for Development locally (Cursor)
1. Clone the repo locally from github on your pc https://github.com/tamerayoub/Lead2Lease
2. Fetch all the branches (git fetch)
3. Create a new branch locally from the branch landing-page-membership (git branch -b omar/new-branch)
4. Ensure to checkout the branch if it doesnt already (git checkout omar/new-branch)
5. Copy the .env file below this message to the root folder path of the repo/app
6. Ensure you have the Node.js v24.10.0 installed
7. Run node -v to check if it installed successfully
8. Ensure you have Npm 6.14.11 installed
9. Run npm -v to check if it installed successfully
10. Open up cursor, and open up the repo
11. run 'npm install' to install all packages
12. run 'npm run dev:cursor' to run app in cursor
